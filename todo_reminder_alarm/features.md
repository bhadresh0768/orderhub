## Short Description
OrderHub helps customers, businesses, and delivery teams place, track, and manage orders in one app.

## Long Description
OrderHub is a role-based order management platform built for customers, business owners, delivery agents, and admins.

From placing orders to tracking delivery and payment status, OrderHub keeps every step organized and visible in real time.

Customers can discover businesses, place detailed orders, and follow order progress. Business owners can manage incoming orders, maintain product catalogs, assign delivery agents, and handle payment updates. Delivery agents can view assigned deliveries and mark orders as delivered with payment collection status. Admins get full moderation tools across users, businesses, orders, support, and app settings.

OrderHub also includes deep links for public business profiles, support ticket workflows, contact forms, local and push notifications, and reporting tools with CSV export.

If you need one app to run multi-role ordering operations from request to fulfillment, OrderHub is built for it.

## Features

### Core Platform
- Role-based experience for `Customer`, `Business Owner`, `Delivery Agent`, and `Admin`
- Firebase-powered backend with real-time updates
- Email/password login and phone OTP login
- Profile onboarding and role-aware home dashboards
- In-app update prompt with store link support
- Internet connectivity status banner

### Customer Features
- Search and filter businesses by name, category, and city
- View public business details and open shareable business links
- Browse business catalog with product variants
- Create orders with:
  - Multiple items
  - Priority selection (`Low`, `Medium`, `Fast`)
  - Scheduled date and time
  - Delivery address details
  - Optional attachments (images/files)
  - Payment method, status, and remarks
- View live order progress and status history
- Edit or delete orders while still in editable state
- Open detailed order pages with itemized billing and GST breakdown

### Business Owner Features
- Register business profile with moderation flow (`Pending`, `Approved`, `Suspended`)
- Maintain catalog structure:
  - Categories
  - Products
  - Variants
  - Variant images from camera/gallery/file
- Place B2B orders to other businesses
- Manage incoming orders:
  - Approve orders
  - Update payment state and collection source
  - Update delivery stages
  - Mark delivered
- Assign or change delivery agent per order
- Share order details (including WhatsApp fallback copy)
- Access order reports and history filters

### Delivery Agent Features
- Delivery dashboard with assigned orders
- View customer delivery details and item summaries
- Mark orders as delivered
- Confirm payment collection at delivery
- Role auto-provisioning by registered delivery phone number

### Admin Features
- Admin panel for operational control
- User management:
  - Edit user info
  - Activate/block users
  - Update subscriptions
  - Approve/reject account delete requests
- Business moderation:
  - Approve, suspend, and review businesses
- Order oversight:
  - Filter and review all orders
  - Edit or delete orders
- Support operations:
  - Track and update support tickets
  - Manage contact-us submissions
- App controls:
  - App update configuration
  - Role-based ads visibility settings

### Order, Payment, and Delivery Tracking
- Structured order lifecycle status tracking
- Delivery status tracking with timestamps
- Payment status tracking (`Pending` / `Done`)
- Payment method and collected-by tracking
- Attachments support for orders and support tickets
- Local notifications for order updates
- Push notifications (FCM) for order lifecycle events

### Reporting and Productivity
- Order history and analytics views
- Filters by status, priority, and date range
- KPI-oriented summary sections
- CSV export via clipboard copy

### Support and Communication
- In-app support ticket submission with attachment support
- In-app contact-us request flow
- Public business profile sharing with deep links and web links
- Copy-ready share text, business links, and deep links

### Profile and Account
- Edit profile details by role
- Upload profile picture and business logo
- Switch eligible role to business owner from profile
- Submit account deletion request for admin approval

### Reliability and Security-Oriented Integrations
- Firebase Authentication, Firestore, and Storage integration
- Firebase App Check activation
- Firestore rules and indexes support for scalable queries
